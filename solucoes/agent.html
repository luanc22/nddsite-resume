NDDPrint Agent

O Agent serve para contabilização e coleta dos dados de impressão.

O NDDPrint Agent pode estar presente no servidor, na estação de trabalho através da notificação para esgotamento de saldo de cotas/políticas ou diretamente no embarcado ligado por USB.

O Agent é responsável pela contabilização lógica, através do Spool do Windows (informações da fila de impressão) e/ou por ERP que contabiliza por usuário.

Também responsável pela contabilização física em dispositivos que funcionam por USB.

O Agent (do servidor/modo completo) é responsável pela aplicação de políticas e cotas que são configuradas no Portal 360.
