NDDPrint Host

O NDDPrint Host serve para monitoração da solução, fica direto no servidor.

  • Serve para monitoração.
  • Parte de autenticação, sincronização de usuários e credenciais.
  • Fazer chamados de Registros do AD (Active Directory)

A instalação do Host pode ocorrer de forma Local, ou na Nuvem.

O nddPrint Host é o que faz o Gateway entre o DataCenter e o cliente (para contabilização dos dados).

*Active Directory: Pasta de gerenciamento da empresa (dados de usuários).
